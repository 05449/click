# 🍪 クッキークリッカー

Cookie Clicker風のクリッカーゲームです。ブラウザで動作し、PWA対応でオフラインでもプレイ可能です。

## 🎮 遊び方

1. `index.html` をブラウザで開く
2. クッキーをクリックしてクッキーを集める
3. 建物を購入して自動生産
4. アップグレードで効率アップ
5. ドラゴンを育てる、転生する、実績を解除する！

## ⚡ 機能

- **8種類の建物** - カーソルから時空の門まで
- **多彩なアップグレード** - 生産効率を強化
- **ゴールデンクッキー** - ランダムに出現するボーナス
- **スキルシステム** - クールダウン付きの強力なスキル
- **庭園** - 植物を育てて報酬を獲得
- **ルーレット・宝箱** - ランダム報酬
- **チャレンジモード** - 時間制限付きの目標
- **マイルストーン** - 達成報酬
- **ドラゴン育成** - 強力なボーナス
- **転生システム** - 天国チップで永続強化
- **デイリーボーナス** - 毎日ログインで報酬
- **PWA対応** - オフラインプレイ可能
- **4種類のテーマ** - ダーク、ライト、クッキー、ネオン

## 📁 ファイル構成

```
├── index.html       # メインHTML
├── style.css        # スタイルシート
├── main.js          # ゲームロジック
├── manifest.json    # PWAマニフェスト
├── service-worker.js # オフライン対応
└── README.md        # このファイル
```

## 🚀 ローカルサーバーで実行

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# VS Code Live Server拡張機能でも可
```

ブラウザで `http://localhost:8080` を開く

## 📱 PWAインストール

モバイルブラウザで開き、「ホーム画面に追加」でアプリとしてインストール可能
